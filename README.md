# pm4

### Usage

```
PM4

Usage

  $ pm4 <command> [options] 

Command List

  delete    Delete an app                                 
  help      Display help information.                     
  restart   Restart an app                                
  save      Save apps list to allow resume on god restart 
  start     Start an app                                  
  status    Get the status of an app or all apps          
  stop      Stop an app                                   

```
